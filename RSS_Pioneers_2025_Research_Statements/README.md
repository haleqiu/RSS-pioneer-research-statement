# RSS Pioneers 2025 Research Statements

Reference research statements from accepted RSS Pioneers 2025 applicants. Used for style analysis when drafting the RSS Pioneers 2026 research statement.

## Download

Requires Python 3.9+ and [openreview-py](https://github.com/openreview/openreview-py):

```bash
pip install openreview-py
python download.py
```

PDFs are saved to this directory. A `manifest.csv` lists title, authors, and filename.

## Usage

These statements are for reference only. Do not commit the downloaded PDFs to version control (this folder is in `.gitignore`).
