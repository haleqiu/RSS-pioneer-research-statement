# RSS Pioneers 2026 Research Statement

RSS Pioneer research statement using the official [RSS paper template](https://roboticsconference.org/docs/paper-template-latex.tar.gz).

## Contents

- `paper_template.tex` – Main research statement (anonymized)
- `references.bib` – Bibliography
- `IEEEtran.cls` – RSS/IEEE document class

## Build

```bash
pdflatex paper_template
bibtex paper_template
pdflatex paper_template
pdflatex paper_template
```

## Submission

- Main content: 2 pages max; references may extend to page 3
- Anonymized per RSS guidelines
- Submit via [OpenReview](https://openreview.net/group?id=roboticsfoundation.org/RSS/2026/Workshop/Pioneers)
